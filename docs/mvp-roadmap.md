# MVP Roadmap

## v0.1 Goals

- load date- and geography-filtered 311 records
- cluster complaint text within a narrow set of top complaint categories
- export map-friendly aggregated outputs
- produce one clear notebook walkthrough for a single district or borough
- document the methodology and parameters used

## v0.1 Non-Goals

- full multilingual support on day one
- exhaustive complaint-type coverage
- polished dashboard UX
- heavy model serving infrastructure

## Stretch Goals

- embedding-based topic grouping
- anomaly detection over rolling time windows
- resolution-gap analysis
- severity or urgency signals from text
- report-card generation

## Release Philosophy

The first release should prove that classical, reproducible NLP can turn a messy civic dataset into something decision-makers and researchers can actually use.
